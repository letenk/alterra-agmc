# How To Run
## Run app with build Docker file
```
docker-compose -f docker-compose.dev.yml up -d
```

## Run with use image from docker hub
```
docker-compose up -d
```

# Docker hub
```
https://hub.docker.com/repository/docker/letenk/altera-bookmarket
```
